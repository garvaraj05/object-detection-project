# Real-Time Object Detection System

A real-time object detection application built using YOLOv8 and OpenCV. The system uses a laptop webcam to detect and classify objects with bounding boxes and confidence scores.

## Technologies Used
- Python
- YOLOv8
- OpenCV
- Computer Vision

## Features
- Real-time webcam detection
- Multi-class object recognition
- Bounding box visualization
- Confidence score prediction
